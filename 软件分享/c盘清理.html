01 软件介绍

Dism++ 是一款 Windows 系统管理优化工具。是一个强大同时轻量易用的小工具，集系统清理、优化、启动项管理、驱动管理等许多常用 Windows 设置与一身，轻点鼠标就可以完成原先繁琐的操作，真正一键全清！是一个 “一站式” 管理优化工具集。

软件为绿色版，体积小，解压即用，也可放在U盘随身携带，方便快捷，比起各种卫士和管家要纯粹不少。Dism++ 的功能和选项十分丰富，同时说明比较清晰，小白用起来也无压力。注意一点就好：不懂的选项保持默认！

02 使用说明

打开解压后的文件夹，选中【Dism++x64.exe】（x64适用于64位系统，x86适用于32位系统，根据自己电脑系统进行选择），然后双击选择Dism++x64.exe即可打开软件。

空间回收

清理 C 盘的垃圾和冗余文件，回收 C 盘空间。点击项目名称，右侧会显示该项的描述和要清理文件的详细信息。不建议频繁清理，一月一次即可。

步骤：空间回收→勾选需要清理的选项，个别会有提示，根据实际情况进行勾选→扫描→清理（时间有时候会很长，请耐心等待）。

优化设置

系统优化选项中，集合了常用的系统自定义设置开关，比如关闭微软小娜、禁用家庭组、启用 win 照片查看器、隐藏任务栏人脉等。这些开关非常具有人性化。

工具箱

工具箱包括了系统备份还原、账户管理、引导修复、映像互转等系统辅助工具。其中像备份还原功能就非常地实用。

系统备份

Dism++能够直接将当前系统备份为WIM、ESD文件，无需进入PE。并且备份还原功能是系统热备份、热还原，也就是说在备份还原的过程中还能正常使用电脑！

第一次备份时需要先创建一个 wim 文件，点击 “浏览”，输入文件名称并选择保存类型 -> 保存，然后选择该文件，即可备份到这个文件中。以后再次备份时，如果 wim 文件已存在，则自动增量备份到现有的 wim 文件中。

操作步骤：工具箱→系统备份→浏览处选择保存路径→确定。在我们重装系统并优化之后，可以进行备份。以备不时之需。

系统还原

需要还原系统时，打开 “恢复功能” 菜单 -> 系统还原，选择 wim 备份文件后，恢复即可。

我们还可以在 “选项 -> 详细设置” 中，启用 “整合引导菜单”，在启动项中添加 Dism++ 选项。当系统无法正常启动时，可通过此选项恢复还原。

操作步骤：工具箱→系统还原→第一个浏览位置选择之前的备份镜像→第二个浏览框选择系统盘位置→点确定。这个在系统出现问题时可以进行还原。

软件还有许多其他的优化设置功能，需要勤劳的小伙伴们发掘更多的姿势去解锁哈！！！
